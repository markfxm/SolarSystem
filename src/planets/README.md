# 行星模型 (Planet Models)

这个文件夹包含了各个行星的类定义。所有行星都继承自 `BasePlanet` 类。

## 基础行星类 (BasePlanet)

`BasePlanet` 类提供了一系列通用功能：
- **网格渲染**：使用共享的单位球体几何体和自定义 Shader。
- **白天/黑夜切换**：自动根据与太阳的夹角切换白天和黑夜贴图。
- **全息模式**：一键切换到 Tron 风格的全息显示。
- **辅助组件**：自动添加经纬度网格和 POI 标记。

## 注意事项

- **贴图**：行星贴图在行星控制器中加载并传递给行星类的 `create` 方法。
- **性能**：通过使用共享几何体和 Shader 材质缓存来优化性能。

# 🌌 Stellar Canvas

> **The Ultimate Celestial Observatory for Your Browser.**

Stellar Canvas is a high-fidelity, interactive 3D Solar System visualization built with Vue 3 and Three.js. Embark on a journey through the cosmos, observe the dance of the planets in real-time, and capture the beauty of the universe with our advanced snapshot engine.

---

## 🚀 The Voyage

Stellar Canvas isn't just a map; it's a window into the mechanics of our solar system. Whether you're a curious star-gazer or a seasoned astronomer, this tool allows you to traverse the void, focus on distant worlds, and manipulate time itself to see how the heavens align.

## ✨ Mission Features

- **🌀 Temporal Distortion**: Manipulate the flow of time. Accelerate through centuries or stay grounded in the present with real-time synchronization.
- **🪐 Planetary Phasing**: Seamlessly navigate between celestial bodies. Focus on the Earth's orbit or zoom out to witness the majesty of the gas giants.
- **📸 The Stellar Moment**: Our crown jewel. Capture stunning, high-resolution (4K) snapshots of the solar system. Apply aesthetic themes like *Cinematic*, *Blueprint*, or *Vintage* to create your own space-themed art.
- **♈ Celestial Mapping**: Toggle the Zodiac ring to see how the sun and planets align with the ancient constellations.
- **🌍 Multilingual Comms**: Full support for both English and Chinese (Mandarin) interfaces.

## 🕹️ Pilot's Guide

Navigate the cosmos with ease using these standard-issue controls:

*   **Left Click + Drag**: Orbit the camera around the current focus.
*   **Scroll Wheel**: Zoom in and out of the celestial stage.
*   **Right Click + Drag / Arrow Keys**: Pan through the stars.
*   **Click on a Planet**: Initiate a focus-lock on that celestial body.
*   **Home Button**: Return to the default Earth-Sun overview.

## 🛠️ Mission Control (Developer Setup)

To launch this project locally, follow these coordinates:

1.  **Clone the Repository**:
    ```bash
    git clone [repository-url]
    cd stellar-canvas
    ```

2.  **Fuel Up (Install Dependencies)**:
    ```bash
    npm install
    ```

3.  **Ignition (Start Development Server)**:
    ```bash
    npm run dev
    ```

4.  **Final Polish (Build for Production)**:
    ```bash
    npm run build
    ```
